# Exercice 4 - BankManager

## Description
Application de gestion de comptes bancaires en Java
utilisant le concept d'héritage.

## Structure du projet
- **CompteBancaire** : classe parente avec les attributs compte et solde
- **CompteCourant** : hérite de CompteBancaire + attribut limite
- **CompteEpargne** : hérite de CompteBancaire + attribut tauxInteret
- **CertificatDepot** : hérite de CompteBancaire + attribut duree
- **Main** : classe principale qui teste les 3 classes enfants

## Comment exécuter
1. Cloner le projet
2. Ouvrir dans IntelliJ
3. Exécuter la classe Main
